Outward Order List


To view Outward Order list, use the "Outward" menu and click the "Add New" button. This will open the new project wizard dialog. Based on Outward Order Type user able to view the order List also order status count display.

The various parts of this dialog are:

  1. Outward Order Type: Specify the type of the Outward Order- Sales Order, Transfer Out, Scrap Order
  2. Order status count display: Outward Order, Allocated, Picking, Quality Check, Packing, Dispatch
  3. Search filter: Search order by Order No, Order Reference No, Group Name, Priority, Delivery Type, Shipping Address, Status
  4. Allocation Plan: System suggest the location, pallet and quantity the SKU need to Pick.
  5. More: contains two options  

5.1 Cancel Order: Before Picking user able to cancel the order.

5.2 De-Allocated: After Allocation user will be able to De-allocate the order.

  1. Outward Order List : This List contains Order No, Reference No, Group Name, Order Date, Expected Delivery Date, Priority, Delivery Type, Shipping Address, Status, Action.
  2. Action : Action contains Document and Communication.

Add New Sales Order

 

To add New Outward Order list, use the “Outward” menu click the "Add New" button. This will open create Sales Order wizard dialog. 

The various parts of this dialog are:

  1. Outward No: After saving the Sales order system will generate a order no i.e outward order number.
  2. Requested Date: By default, system will take current date.
  3. Exp. Delivery Date: It is a expiry date of the order to be delivered.
  4. Reference No: Every order should have unique reference no.
  5. Delivery Type: Contains two type Standard and Urgent.
  6. Client: When clicked on search icon clients name will display. User need to select one client.
  7. Bill To Address: After client selection by default Address should display.
  8. Ship To Address: After client selection by default Address should display.
  9. Priority: Order contains 3 priorities- Routine, Urgent, Very Urgent.
  10. Created By: Login User name will display.
  11. Remark: User can  enter Remark text if any.
  12. Add SKU: When clicked on Add SKU, SKU list will be displayed. Need to select SKU from list. After selection SKU Details will display.
  13. Requested Quantity: Against SKU user need to enter Requested Quantity and click on save SKU button. Same process needs to repeat for all SKUs.
  14. Save:  After adding SKU in order user need to click on Save button.


Sales Order will be created and displayed in Outward Order List of Sales Order.



Add New Transfer Out Order 

To add New Transfer Out Order, use the “Outward” menu select the Order Type as Transfer Out then click the "Add New" button. This will open create Sales Order wizard dialog. 

The various parts of this dialog are:

  1. Outward No: After saving the order system will generate Outward No.
  2. Requested Date: By default, system will take current date.
  3. Exp. Delivery Date: When clicked on text box system will display calendar from that user will be able to select the date..
  4. Reference No: Every order should have unique reference no.
  5. Delivery Type: Contains two types Standard and Urgent.
  6. Warehouse: When clicked on search icon warehouse name will display expect warehouse display in filter. User need to select one warehouse.
  7. Bill to Address: After warehouse selection by default Address should display.
  8. Ship to Address: When clicked on search icon address list should display. User need to select one address.
  9. Priority: Order contains 3 priorities- Routine, Urgent, Very Urgent.
  10. Created By: Login User name will be displayed.
  11. Remark: User can enter Remark text if any.
  12. Add SKU: When clicked on Add SKU, SKU list will be displayed. Need to select SKU from list. After selection SKU Details will be displayed.
  13. Requested Quantity: Against SKU user need to enter Requested Quantity and click on save SKU button. Same process needs to repeat for all SKUs.
  14. Save:  after adding SKU in order need to click on Save button.



Transfer Out Order will be created and display in Outward Order List of Transfer Out Order.
