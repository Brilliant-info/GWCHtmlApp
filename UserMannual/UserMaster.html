User Master

User Master 

To create user, use the "Administrator" menu and click the "User Master" button. This will open the user list wizard dialog. 

The various parts of this dialog are:

  1. Search Filter: Search User by User Type, First Name, Last Name, Employee No., Email ID, Mobile No., User Name.
  2. User Type: User types are-Super Admin, Admin, Warehouse Management, Inward, and Outward.
  3. User List: User List contains User Type, First Name, Last Name, Employee NO, Email Id, Mobile No, and User Name.
  4. Active/Inactive:  If it is checked means user is available/Active, & if it is Unchecked Means User is not available/INACTIVE.
  5. Action: Action contains –Save, Lock/Unlock, Edit, Role Master, Add Warehouse, Add Customer, add Client 
    1. Save: Click on save button to save the data.
    2. Lock/Unlock: If the 'Lock ICON' is lock means user can’t login again & if it is Unlock means user can login again.
    3. Edit: User can use this button to edit filled information.
    4. Role Master: User can assign the role to user from the drop-down as Admin,warehouse manager.As per selected role user should select the menus from list by clicking on action button.
    5. Add Warehouse: User can assign warehouse.For assigning warehouse user need to click on add warehouse from list & need to select the warehouse & click on submit button then the warehouse will display in the warehouse list.
    6. Add Customer: User can assign customer.
    7. Add Client: User can assign client.

Added User Data will display in User List.

Role Master

To Assign the roles click on Role Master select the role from dropdown. This will open Role Master wizard dialog. 

The various parts of this dialog are:

  1. Select Role: Select the role.
  2. Menu List: Select the menu from the menu list to give the access of that menu to the user. 
  3. Select all: If you want to give all menu access to the role you can select all roles by click on Select all. 
  4. Save icon: Click on save icon to save the operation.
  5. Cross: Click on Cross icon to close the page.


Role will be assign to the user.


Add Warehouse

To add the warehouse, Click on Add warehouse icon  This will open the Warehouse List wizard dialog. Select the warehouse and click on Submit Button.

The various parts of this dialog are: 

  1. Warehouse List: Warehouse List contains Warehouse Code, Warehouse Name. 
  2. Submit: Click on submit button to add the warehouse. 
  3. Action: There is a cross icon you can remove the warehouse by clicking on Cross icon. 

Warehouse will get assign to user. 

Add Customer

To add the Customer, Click on Add Customer  This will open the Customer List wizard dialog. Select the Customer and click on Submit Button.

The various parts of this dialog are:

  1. Customer List: Customer List contains Customer Code, Customer Name.
  2. Submit: To add the customer.
  3. Action: There is a cross icon you can remove the added customer by clicking on cross icon.

Customer will get assign to user.

Add Client


To add the Client, Click on Add Client icon.This will open the Client List wizard dialog. Select the Customer and click on Submit Button.

The various parts of this dialog are:

  1. Client List: Client list contains Client code, Client Name, SAP Code.
  2. Submit: To submit the client.
  3. Action: There is a cross icon you can remove the added customer by clicking on cross icon.

Customer will assign to selected user.
