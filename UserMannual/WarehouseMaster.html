Warehouse Master

Warehouse Master 

To add a warehouse , use the "Administrator" menu and click the "Warehouse Master" button. This will open the Warehouse list wizard dialog. 

The various parts of Warehouse list dialog are:

  1. Search Filter: Search User by Warehouse Name, warehouse Code, Type.
  2. Add New: Click on Add New button to add the new Warehouse.
  3. Warehouse Code: Every Warehouse Code should be unique.
  4. Warehouse Name: Enter warehouse Name in the Field.
  5. Type: Specify the type of the Warehouse- Standard, Virtual.
  6. Remark: If you want to add any additional information so you can write in the remark.
  7. Active: Checked (Yes) - Means available to use.  Unchecked (No) - Not available to use.
  8. Action: 

1. Edit- User can able to edit customer info.

2. Contact Info- Click on contact icon to add contact information.

3. Location- User can add location information.

Contact Info

To add the contact information Click on contact icon which is in action column. It will open contact information list wizard dialog.

The various parts of the dialog are:

    1. Fill all data in empty fields like Contact Name, Email ID, Mobile No., Address, Country, State, City, Postal code.
    2. Active: Checked (Yes) - available.  Unchecked (No) - Not available.
    3. Action: Save- Click save icon to save the information.

             Edit- Click edit icon to edit the information.

Contact information will be added in the list.

Location



To add the location information Click on location icon which is in action column. It will open Location list wizard dialog.

The various parts of the dialog are:

  1. Last Sort Code: Display the latest sort code inserted.
  2. Search Filter: Search location by Location type, Zone location, Sort code, Location code, Shelf, Section, Pathway, Floor, and Building.  
  3. Create Cycle Count Plan: To match System quantity with Warehouse it is done by Cycle count.        

To create Cycle Count Plan by Location, use the "Administration" menu. This will open the Administrator wizard dialog and click on Warehouse Master Button then click the location icon against warehouse then click on Create Cycle Count Plan button. 

The various parts of this dialog are:

  • Before clicking on Create Cycle Count Plan button user need to select single or multiple location whose Cycle Count need to be perform
  • Plan Title: Need to enter the Plan Title for the Cycle Count to perform location wise.
  • Plan ID: Need to enter the Plan ID for the Cycle Count to perform location wise. This field value is unique.
  • Submit. When click on Submit Cycle Count Plan for location will be created.


Cycle Count Plan created will display in Administration menu of Cycle Count Section.


  1. Location Type: Specify location type like store, Rejection, Loading, Unloading etc. - These are different type of location in warehouse.
  2. Zone Location: These are different type of zone in warehouse.
  3. Sort code: Using sort code nearest location is searched in warehouse.
  4. Location code: Name of the location.
  5. Shelf: Different shelf of section used to store goods in warehouses.
  6. Section: Different section of pathway used to store goods in warehouses.
  7. Pathway: Different pathway of floor used to store goods in warehouses.
  8. Floor: Different floor of building used to store goods in warehouses.
  9. Building: Different building name of warehouse.
  10. Capacity In: Unit of capacity used to store goods in warehouses.
  11. Capacity: Capacity of location.
  12. Active: Checked (Yes) - Means available.  Unchecked (No) - Not available.
  13. Action: Save- Click save icon to save the information.

             Edit- Click edit icon to edit the information.

Location information will be added in the list.
